## Types

### SaveContent

```
SaveContent {
  objects: SavedObject[]
}

SavedObject {
  id: UUID
  name: string              // e.g. Unity GameObject hierarchy name
  data: SavedObjectData[]
}

SavedObjectData {
  key: string
  value: string
  type: string              // class to cast value to on-load, e.g. System.Boolean in C#
}
```
